# ENS Brick Identity

[![Synthesis Submission](https://img.shields.io/badge/Synthesis-Submit-blue?logo=gitbook)](https://synthesis.devfolio.co/projects/ens-brick-identity-fe6d)

Human-readable agent addressing using ENS names. Instead of raw hex addresses, agents can be addressed as brick.eth.
